Azure Data Lakehouse Interactive Architecture

Esta é uma ferramenta interativa desenvolvida em React para explicar a arquitetura de um Data Lakehouse na nuvem Azure.

📁 Estrutura de Pastas

Para rodar localmente, organize os arquivos da seguinte forma:

/
├── index.html
├── package.json
├── tailwind.config.js
├── postcss.config.js
├── README.md
└── src/
    ├── main.jsx
    ├── App.jsx
    └── index.css


🚀 Como Executar

Certifique-se de ter o Node.js instalado.

No terminal, dentro da pasta do projeto, instale as dependências:

npm install


Inicie o servidor de desenvolvimento:

npm run dev


Para gerar a versão final para o GitHub/Vercel:

npm run build


☁️ Deploy no GitHub/Vercel

Crie um repositório no GitHub e suba os arquivos.

Acesse vercel.com.

Conecte seu GitHub e selecione este repositório.

A Vercel detectará automaticamente as configurações do Vite e publicará o site.
